# Bot Consulta Resolución ANM 371/2024

Bot inteligente para consultar información sobre la Resolución 371 de 2024 de la Agencia Nacional de Minería de Colombia.

## Instalación

1. Clona o descarga el proyecto
2. Instala dependencias: `npm install`
3. Configura tu API key de OpenAI en el archivo `.env`
4. Ejecuta: `npm start`

## Uso

El bot responde preguntas específicas sobre la Resolución 371/2024, incluyendo:
- Cronogramas de implementación
- Formatos de registro obligatorios
- Componentes tecnológicos
- Sanciones y obligaciones

## Tecnologías

- Node.js + Express
- OpenAI GPT-3.5
- HTML/CSS/JavaScript
